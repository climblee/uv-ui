## ActionSheet 操作菜单

> **组件名：uv-action-sheet**

本组件用于从底部弹出一个操作菜单，供用户选择并返回结果。

本组件功能类似于uni的uni.showActionSheet API，配置更加灵活，所有平台都表现一致。
