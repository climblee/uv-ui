## QRCode 二维码生成器

> **组件名：uv-qrcode**

超级强大二维码生成组件，可扩展性高。自定义二维码样式，如随机颜色、圆点、方块、块与块之间的间距等等，具体请在下方查看文档。

灵活配置，开箱即用，文档全面，支持自定义二维码风格。
